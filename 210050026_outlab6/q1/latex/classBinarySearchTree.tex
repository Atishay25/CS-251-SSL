\doxysubsection{Detailed Description}
Binary Search Tree Data Structure. 

BST is a Binary Tree Data Structure in which -\/
\begin{DoxyEnumerate}
\item Left Subtree of each Node stores Nodes with values lesser than the Node
\item Right Subtree of each Node stores Nodes with values greater than the Node
\item Left and Right Subtree each must also be a Binary Search Tree 
\end{DoxyEnumerate}
