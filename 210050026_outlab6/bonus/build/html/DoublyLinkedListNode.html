DoublyLinkedListNode module¶

class DoublyLinkedListNode.DoublyLinkedListNode(data)[source]¶

Bases: object

Node of a Doubly Linked List
A Double Linked List Node stores an additional pointer as compared to a Singly Linked List Node, mainly to point to Previous Node as well
Each Node in a Singly Linked List stores -
  1. Data value (can be of any data type like int, str, float etc)
  2. Pointer (or reference) to its next Node
  3. Pointer (or reference) to its previous Node
Parameters:
  • data (int) – Value to be stored in a Node
  • next (SinglyLinkedListNode) – Pointer to next Node
  • prev (DoublyLinkedListNode) – Pointer to previous Node
__init__(data)[source]¶
Constructor to initialize a new Doubly Linked List Node object
It initializes next and prev to None and data to given data value
Parameters:

data (int) – Value to be stored in Node
Example:
>>> dlNode = DoublyLinkedListNode(45)
>>> print(dlNode.data)
45
