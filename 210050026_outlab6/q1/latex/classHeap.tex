\doxysubsection{Detailed Description}
This class implements the \mbox{\hyperlink{classHeap}{Heap}} Data Structure. 

A \mbox{\hyperlink{classHeap}{Heap}} is a special Tree-\/based data structure in which the tree is a Binary Tree that stores priorities (or priority-\/element) pairs at nodes.
